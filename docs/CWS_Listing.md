# Chrome Web Store — listing NativeFill

**Nazwa**: NativeFill — Native‑like Form Autofill  
**Krótki opis**: Open‑source, privacy‑first form autofill with native‑like suggestions and a right‑click menu.

**Pełny opis (skrót):**  
- **Sugestie „native‑like”** w polach tekstowych (klawiatura, ARIA).  
- **Menu kontekstowe** (PPM) z folderami.  
- **Profile, tagi, aliasy**, import/eksport JSON.  
- **Reguły per domena** i opcjonalne **wyłączenie na hostach**.  
- **WASM fuzzy‑match** (szybko), fallback TS.  
- **Multi‑browser**: Chrome/Edge/Firefox/Safari.  
- **Prywatność**: dane lokalne; opcjonalny Sync/Backup; **brak telemetrii**.
